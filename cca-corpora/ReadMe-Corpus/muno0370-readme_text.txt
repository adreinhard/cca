Adventure!                          The original great game!  (As I modified it when converting it from Honeywell FORTRAN 77 to MS DOS PDS FORTRAN v5.10).Executable game files on ADVENT.ZIP      ADVENTUR.EXE - Adventure Program     AD.DAT       - Adventure Database -- must be with program file     README.ADV   - This file.     FILE_ID.DIZ  - BBS Directory Description File.Now, if you want to take all of the fun and all of the challenge out of thegame, here's everything you need to modify, figure out, and maintain Adventure.Of course, you need MS DOS PDS FORTRAN v5.10 or greater.Source code on ADVSRC.ZIP      ADVENTUR.FOR - Adventure Program Source Code in MS DOS FORTRAN 77     ADVEDAT.ASC  - ASCII Input for Database Construction     DB.FOR       - Database Construction Program source, reads ADVEDAT.ASC     DB.EXE       - Database Construction Program.     DB.TXT       - Database Construction Output File.     ADVEMAP.FOR  - How to print a cave cheat chart Source Code     ADVEMAP.EXE  - Cave cheat chart program.     ADVEMAP.TXT  - Cave cheat chart output file.     ADARR.FOR    - Dump AD.DAT in Array Format Source Code     ADARR.EXE    - Dump AD.DAT program.     ADARR.TXT    - Dump AD.DAT output file.     AD2DUMP.FOR  - Dumps Arrays from ADVENTUR.SV File Source Code     AD2DUMP.EXE  - Dump Arrays from .SV program.     AD2DUMP.TXT  - Dump Array output file     ADV.BAT      - Batch file to compile, link, and execute.     PERFECT.TXT  - A walk through a perfect 370-point game.     README.ADV   - This file.     FILE_ID.DIZ  - BBS Directory Description File.Have a good time with this!  If you have questions, I'll give it a shot, but you can thank the legendaryWillie Crowthers for making this one of the oldest and best computer games,and one of the most widely ported in the industry!ษอออออออออออออออออออออออออออออออออออออออออออออออออออออออออออออออออออออออออออออปบ                  Adventure - The Original Colossal Cave.                    บฬอออออออออออออออออออออออออออออออออออออออออออออออออออออออออออออออออออออออออออออนบ                  Written 1990-1996 by Paul Muคoz-Colman.                    บ บ    Member, Association of Shareware Professionals (ASP).
