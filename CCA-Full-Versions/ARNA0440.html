Adv440 downloads

This is Peter Luckett's and Jack Pike's Adv400, dated 31 Dec 1978.

  • Adv440 was originally known as Adventure II. In deference to this initial name, file naming conventions of the game are based on the AdvII name-stem. A separate document describes the history of the game.
  • When unpacked, the playable versions consist of an executable and a data file, both of which must be in the same directory.
  • The game is fundamentally "windows unaware" and has to be played in a console window.
