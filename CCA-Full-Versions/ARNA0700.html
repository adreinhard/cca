This superset of the original Adventure incorporates my Adv660 expansion of the game. See Adv770 FAQ (04 December 2017) for answers to some questions about the game. (I said a FAQ! – not a walkthrough or a hint sheet.) There is also a separate bug list (01 December 2018), which lists known bugs and records their fixing.

Play it now

You can play Adv770 right now, either on a cloud server or, if you are using a robust modern browser, on your local machine (6.7 MB).
 
The local version works well with up-to-date Firefox, Chrome, Safari and MS Edge, though other browsers may not cope and some ad- and popup-blockers block it for some reason. It runs strictly locally, with no further network access – games are saved in your browser's local storage space. You can save the page locally too, and later just point the same browser at the saved html file to continue playing.

Pre-built versions

 
Thanks to the magic of Emscripten, there is now a pure HTML/Javascript version which in theory should run locally on your machine in any robust modern browser. It works fine in up to date Firefox, MS Edge, Safari and Chrome (including tablets, though on smaller ones virtual keyboard may take too much screen space), but note that some ad- and popup-blockers take dislike to it for some reason. You can either kick it off straight away via adv770.html (6.7 MB) and then save the page at any stage, or you can download the zipped version adv770h.zip (1.4 MB), unzip it and point your browser at the resulting adv770.html file.
A unified Linux console/browser build adv770.tgz (820.3 KB) with command editing, defaulting to browser-based display (but if you are using Firefox or Chrome, you may prefer the much simpler HTML/Javascript version).
Mac OS X (10.4 upwards) (Intel ony) Adv770.dmg (1.9 MB) defaulting to browser display.
PowerPC Macs can use universal build Adv770ppc.dmg (5.2 MB) frozen at version 2.12 – sorry, I am no longer able to build PPC executables.
Brian Ball's port of Adv770 for iPad (iOS 6.0 or later) is available as a free iTunes download from
https://itunes.apple.com/us/app/id988054922.
On Windows 10 you need to use the new MS Edge browser with the HTML/Javascrip version of the game. You can either kick it off straight away via adv770.html (6.7 MB) with command editing, and then save the page at any stage, or you can download the zipped version adv770h.zip (1.4 MB), unzip it and point your browser at the resulting adv770.html file.
Windows installer adv770w.exe (1.1 MB) for an executable capable of browser or console display – the display choice is offered as a part of the installation process.
Zipped windows-unaware MSDOS executable adv770.zip (788.8 KB) with no command line editing. For use only on true DOS machines or under DOS emulators (e.g. DOSBOX).
Console mode can be forced by adding -C to command line invocation. See the bundled README file for more details.

A-code sources

The gzipped tarball adv770a.tgz (366.2 KB) contains the A-code sources of Adv770. Follow this link for an explanation of the process of creating a game executable from its A-code source. Even if you don't need things being explained to you, you may still need to pick up the the C sources of the acdc translator and of the A-code kernel. Both are to be found within this gzipped tarball: acode-12.74.tgz (1.0 MB)

Derived C sources

The gzipped tarball adv770c.tgz (784.4 KB) contains the ANSI C source files and header files, derived by my acdc A-code-to-C translator from the Adv770 A-code source. The basic build is trivial: just compile and link the lot using an ANSI C compiler, and you'll have a fully functional adv770 executable, defaulting to browser display. However, in the console mode (as opposed to the browser one) this executable won't support command history or editing, nor will it be windows-aware. Follow this link if you need instructions for building less "dumb" console executables from the same C source.
