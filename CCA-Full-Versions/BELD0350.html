Description

This is a C language port of Crowther's original ADVENTURE. In order to restore the original version (which was then thought to be lost) Marvin W. Rasmussen started out with the BSD-C port of Adventure from the CP/M public domain library, made by Jay R. Jaeger and L. C. Calhoun and compared it to two older versions written in Fortran: an ISIS-II Fortran-77 version (itself merged from a Fortran-10 version made by Crowther and Woods and a Fortran-IV-Plus version called ADVENTURES made by Kent Blackett at DEC) and a version written for an unknown mini computer (probably a Harris). Using the Fortran versions, the original wording (with spelling mistakes and all) and interpreter mechanisms were reintroduced into the new C version written for the Atari ST.

This was for many years regarded as as close to the original Crowther/Woods ADVENTURE one could get, until an original Fortran version of Crowther's ur-Adventure was discovered on a backup tape in 2007.
